# 闫雨浪 - 付费聊天社交软件 MVP

> 仿识俏社风格的付费聊天社交应用，男用户花钱聊天，女用户回复赚钱

## 🎯 项目介绍

闫雨浪是一个模拟真实社交付费场景的 MVP 产品，核心逻辑是：
- **男用户** 发消息需要支付金币（按条扣费）
- **女用户** 回复消息获得金币（平台抽30%）
- 通过动态广场引流，女用户发自拍吸引男用户付费私聊

## ✨ 功能特性

### 用户端
- 📱 手机号注册登录（模拟验证码）
- 👤 完善个人资料、头像上传、兴趣标签
- 🏠 首页推荐卡片（女用户优先展示）
- 💬 实时聊天（SocketIO）
- 💰 金币充值/消费/提现
- 📷 动态广场（图文发布、点赞、评论）
- 👁️ 访客记录
- ⚙️ 个人设置

### 核心付费逻辑
```
男用户发消息 → 检查余额 → 扣金币 → 发送消息
                               ↓
                         通知女用户获得收入（扣除30%平台抽成）
女用户回复 → 计算收益 → 收入到账
```

### 后台管理
- 用户列表
- 动态管理
- 数据统计

## 🛠️ 技术栈

- **后端**: Flask + Flask-SocketIO + SQLAlchemy
- **数据库**: SQLite
- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **实时通信**: Socket.IO
- **图片处理**: Pillow

## ⚠️ 注意事项

1. **这是MVP测试版**：充值和提现都是模拟的，不涉及真实支付
2. **验证码是模拟的**：任意4位数字均可
3. **图片上传限制**：单张最大5MB
4. **首次运行**：会自动创建数据库和测试数据
5. **SocketIO**：需要浏览器支持WebSocket
